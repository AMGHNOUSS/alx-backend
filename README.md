# ALX Backend

Module of **Alx Backend** , carried out during **Full Stack Software Engineering studies** at **[ALX School](https://www.alxafrica.com/)**.

## Technologies
* Scripts written in Bash 5.1.16
* Tested on Ubuntu 20.04 LTS
* Python 3.11.9
* Node v18.19.1
